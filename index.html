Definición del Problema

Este espacio está destinado para describir un problema real que pueda resolverse con métodos numéricos.

Métodos Numéricos

  • Interpolación y ajuste de curvas
  • Solución de ecuaciones no lineales
  • Integración y derivación numérica
  • Resolución de ecuaciones diferenciales
  • Optimización numérica

Interfaz de Usuario

Este componente mostrará los campos de entrada para datos, los resultados y visualizaciones. Será desarrollada usando tecnologías como HTML, CSS, JS, y conectada a Matlab/Octave mediante API.

Método Secante

Resuelve la ecuación f(x) = sin(x) + 3cos(x) usando el método de la secante.
